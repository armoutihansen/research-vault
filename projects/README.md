# Project notes (Layer 3)

One note per research project: `<slug>.md`. Lifecycle state is the `status:` frontmatter property
(`feasibility → planned → ongoing → finished`, with `on-hold` and `rejected`). Created by
`/promote-idea`; transitions made only by `/project-status` (which logs them to the Decision log).

dataview/kanban render the project board from `status`.

See [`../docs/architecture.md`](../docs/architecture.md) and [`../CONTEXT.md`](../CONTEXT.md).
